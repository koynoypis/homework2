Ικανότητες/Εμπειρία

Η εμπειρία μου πηγάζει κυρίως απο τις διάφορες εργασίες(ομαδικές ή ατομικές) που έχω ολοκληρώσει κατα την διάρκεια των σπουδών μου

Αντικείμενο εργασιών:
  • Προγραμματισμός σε C
  • Προγραμματισμός σε Java
  • Προγραμματισμός σε επίπεδο συστήματος(MINIX)
  • Προγραμματισμός σε επίπεδο υλικού(assembly,vhdl)
